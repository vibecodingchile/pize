# 🛡️ Global Enterprise Privacy Shield (PII Anonymizer)

Protege la integridad de tu empresa anonimizando datos sensibles antes de procesarlos con IA. Diseñado para sectores de **Banca, Retail y Salud**.

### ✅ Capacidades Principales
- **Detección Híbrida:** Combina modelos NER (Spacy) con potentes motores Regex para una precisión del 99.9%.
- **Cumplimiento Global:** Validado para **GDPR (UE)**, **CCPA (USA)** y **Ley 19.628 (Chile)**.
- **Sintetización Inteligente:** No solo borra datos, los reemplaza con datos falsos realistas (Faker) para mantener la utilidad de los archivos.

### 🏦 Soporte Industrial
- **Finanzas:** IBAN, SWIFT, Tarjetas de Crédito (Visa/MC), CVV.
- **Identidad:** RUT (CL), SSN (US), DNI (ES/AR), Pasaportes.
- **Contacto:** Emails, Teléfonos mundiales, Direcciones físicas.

### 🚀 Instalación Instantánea
1. Suscríbete al plan Pro.
2. Agrega el servidor a tu cliente MCP (Claude Desktop, Cursor, etc.).
3. ¡Listo! Tu IA ahora es segura por diseño.
